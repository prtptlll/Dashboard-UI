#  Modern SaaS Dashboard UI

A clean, responsive SaaS admin dashboard built with a focus on UI design, interactivity, and real-world SaaS layout structure.

##  Features

* Modern SaaS dashboard layout with sidebar navigation  
* Fully responsive design (mobile, tablet, desktop)  
* Interactive sidebar with active state switching  
* Analytics chart integration using Chart.js  
* Recent activity table with hover effects  
* Smooth UI animations and hover interactions  
* Clean Tailwind CSS utility-based styling  

##  Tech Stack

* HTML  
* Tailwind CSS  
* JavaScript  
* Chart.js  

##  Purpose

This project demonstrates:

* Real-world SaaS dashboard UI structuring  
* Frontend component-based thinking  
* Responsive layout design using Tailwind CSS  
* JavaScript-based UI interactivity  
* Dashboard-style data visualization integration
